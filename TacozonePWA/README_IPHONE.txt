Tacozone PWA Demo (iPhone) ✅

هذه نسخة تجريبية (POS + KDS + Dashboard) تعمل على iPhone وحده (بدون PC الآن).
بعدها نربطها مع الكمبيوتر الحقيقي في المطعم عبر Wi‑Fi.

تشغيلها على iPhone 12 عبر GitHub Pages:
1) أنشئ Repo جديد (مثلاً: TacozonePWA)
2) ارفع محتوى هذا المجلد إلى الـRepo (Upload files)
3) فعّل GitHub Pages:
   Repo -> Settings -> Pages
   Source: Deploy from a branch
   Branch: main / root
   Save
4) سيظهر رابط Pages، افتحه على Safari
5) Share -> Add to Home Screen

البيانات تحفظ داخل iPhone (localStorage).
